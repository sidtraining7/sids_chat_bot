## Architecture Diagrams
We have provided the architecture diagram in two formats:

### 1. Mermaid (`architecture.mermaid`)
This is a standard text-to-diagram format supported by GitHub and VS Code.
*   **To View**: Install the "Markdown Preview Mermaid Support" extension in VS Code, or view it directly on GitHub.
*   **To PDF**: Open the preview in VS Code -> Right Click -> "Print to PDF".

### 2. Draw.io (`architecture.drawio`)
This is an editable diagram file.
*   **To View/Edit**:
    *   **VS Code**: Install the "Draw.io Integration" extension by Henning Dieterichs.
    *   **Web**: Go to [app.diagrams.net](https://app.diagrams.net/) and open this file.
*   **To PDF**: Open the file (in VS Code or Web) -> File -> Export as -> PDF.
